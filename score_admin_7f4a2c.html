🏕️ 아이돌 수련회 스코어보드 관리자

이 페이지는 일반 이벤트 페이지에 노출되지 않습니다. 관리자 키가 맞아야 공유하기 점수를 입력할 수 있습니다.

관리자 키는 Supabase SQL 실행 시 발급된 값을 사용하세요.
이 관리자 페이지에는 키가 저장되어 있지 않습니다.

점수 입력

점수 규칙: 다짐 1개=10점, 좋아요 1개=1점, 공유하기 1회=50점입니다.
일반 페이지 점수판은 1분마다 자동 업데이트됩니다.
공유하기 점수는 매일 오전 9시 59분(한국시간)에 업데이트하는 운영 기준입니다.
수기 입력은 즉시 스냅샷, 자동 집계 점수는 30분 단위로 스냅샷 저장됩니다.
공유하기 공식 점수는 버튼 클릭 수가 아니라 관리자 검수 완료 수 기준으로 넣는 걸 권장합니다.
